# Interactive Portfolio Website 🚀

## 📌 Project Description
This is a JavaScript-based interactive portfolio website that includes dynamic features like dark mode, form validation, image slider, and a to-do list.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ✨ Features
- Show / Hide content
- Image slider
- To-do list (add/remove)
- Contact form validation
- Dark / Light mode
- Local Storage support

## 📂 Project Structure
- index.html
- style.css
- script.js
